# **App Name**: FeeFlow

## Core Features:

- Welcome Page: Landing page with school logo, name, and a 'Get Started' button to initiate the fee check process.
- OTP Verification: Implement SMS OTP verification using the provided API to verify the user's phone number. Uses the OTP tool, to send a message to the end user containing the secret value to enter into the web page.
- Class Selection: Display available classes in a grid format for selection.
- Student Selection: Display students in a table format for the selected class, allowing the user to choose a student.
- Relationship Form: A modal/form to capture the relationship of the user to the selected student.
- Fee Status Display: Display fee status including total fees, amount paid, balance, and due date fetched from the Google Sheet.
- Print/Download Result: Provide options to print or download the fee status as a PDF.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to evoke trust and stability, inspired by the app's educational context.
- Background color: Very light indigo (#F9FAFB) to provide a clean and calm backdrop.
- Accent color: Violet (#8B5CF6) to create contrast for interactive elements, fitting with the app's tech-forward purpose.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text.
- Feather icons for a simple and consistent UI.
- Use a clean, card-based layout with ample white space to avoid clutter.
- Subtle transitions and animations to provide feedback on user interactions.
